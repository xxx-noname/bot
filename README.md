## INSTALLATION

```bash
> pkg update && pkg upgrade
> apt-get update
> apt-get upgrade
> pkg install git
> pkg install bash
> git clone https://github.com/MrK4ZUT0/botwa
> cd botwa
> bash install.sh
> node index.js
```

Scan the qr code that appears, using whatsapp web!
